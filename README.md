# Sandwich Shop App

A Flutter application for ordering sandwiches with customizable options.

## Features

- Choose sandwich size: **six-inch** or **footlong**
- Choose bread type: **white**, **wheat**, **wholemeal**
- Add custom order notes (e.g., “no onions”)
- Increase or decrease quantity (with max limit)
- Real-time visual order summary
- Clean UI using Flutter Material Design
